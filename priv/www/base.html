Adding the repository

First, add the repository public key for apt-secure using following commands :

 wget {{ url }}/{{ repository }}/{{ pubkey }} 
 sudo apt-key add {{ pubkey }}

Add line(s) like below to your /etc/apt/sources.list file :

 deb {{ url }}/{{ repository }} SUITE COMPONENTS
